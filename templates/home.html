Background

The original dataset had a set of 500 observations. While doing EDA (Exploratory Data Analysis), we found that some wind values were generating zero power. We figured that these observations were due to maintenance or fault and so we dropped them. Doing some into wind power generation, we found that there is a cut off for power generation in wind turbine where wind speeds can be too high and can damage the motor, hub and blades and so the upper limit of wind speed exist. We have chosen 100 and a top speed.
